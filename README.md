# 機場接送服務網站

一個現代化、響應式的機場接送服務網站，採用白色簡約設計風格。

## 特色功能

### 🚀 立即預約
- **LINE預約按鈕**：位於頁面頂部，點擊可直接跳轉到LINE聊天
- **電話預約按鈕**：一鍵撥打電話進行預約
- 預約按鈕固定在頁面頂部，方便用戶隨時預約

### 🎨 設計風格
- **白色簡約風格**：清爽的白色背景，簡潔的設計元素
- **現代化UI**：使用Inter字體，圓角設計，柔和的陰影效果
- **漸變色彩**：藍色漸變作為主要強調色，綠色和橙色用於預約按鈕

### 📱 響應式設計
- 完全響應式設計，支援所有設備尺寸
- 移動端優化的導航菜單
- 觸控友好的按鈕和表單元素

### ✨ 互動效果
- 平滑滾動動畫
- 懸停效果和過渡動畫
- 滾動觸發的動畫效果
- 表單驗證和通知系統

## 技術特點

### 前端技術
- **HTML5**：語義化標籤，無障礙設計
- **CSS3**：Flexbox和Grid佈局，CSS變數，動畫效果
- **JavaScript (ES6+)**：模組化代碼，事件處理，動畫控制

### 設計系統
- **色彩方案**：
  - 主色：#0072ff（藍色）
  - 輔助色：#00c6ff（淺藍）
  - 成功色：#00b900（綠色）
  - 警告色：#ff6b35（橙色）
  - 文字色：#1a202c（深灰）

- **字體系統**：
  - 主要字體：Inter
  - 字重：300, 400, 500, 600, 700

- **間距系統**：
  - 基礎間距：20px
  - 區段間距：100px
  - 卡片內邊距：30-40px

### 響應式斷點
- **桌面端**：> 768px
- **平板端**：≤ 768px
- **手機端**：≤ 480px

## 自定義配置

### 聯絡資訊
在 `index.html` 中修改以下資訊：
```html
<!-- LINE ID -->
<a href="https://line.me/R/ti/p/@airporttransfer">

<!-- 電話號碼 -->
<a href="tel:+886912345678">

<!-- 電子郵件 -->
<p>info@airporttransfer.com</p>
```

### 價格方案
在 `index.html` 的價格區段中修改價格和服務內容。

### 公司資訊
在 `index.html` 的關於我們和頁腳區段中修改公司相關資訊。
